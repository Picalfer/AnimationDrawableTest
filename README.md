# AnimationDrawable Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Animation](https://img.shields.io/badge/Feature-AnimationDrawable-orange.svg)](https://developer.android.com/reference/android/graphics/drawable/AnimationDrawable)

Тестовое приложение для изучения AnimationDrawable в Android. Демонстрирует создание анимированного фона с настраиваемой длительностью.

## Функциональность

- Анимированный фон с плавными переходами
- Настройка времени затухания (fade duration)
- Запуск анимации программно
- Работа с AnimationDrawable

## Технологии

- Kotlin
- AnimationDrawable
- ConstraintLayout
- Background animation
